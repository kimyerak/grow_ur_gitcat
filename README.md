<aside>
🗣 CitHub는 개발자들을 위한  아바타 기반의 관계망 확장, 셀프 기록 서비스입니다.

- GitHub API를 이용해 나의 커밋 기록을 불러오고 이를 기반으로 다양한 이벤트가 일어납니다.

</aside>

### **개발 환경**

- **FrontEnd**: React
- **BackEnd:** Nest.js
- **DB:** MongoDB
- **Cloud:** Kcloud

- 그동안의 커밋 기록이 반영됩니다.
